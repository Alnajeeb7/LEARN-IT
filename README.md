# Data Science Practice System

Interactive question-based learning system for mastering data science concepts.

---

## Overview

This project is a structured study tool designed to help users learn and revise data science topics through practice questions, flashcards, and quizzes. It focuses on active learning and progress tracking.

---

## Features

- Organized question bank by topics  
- Flashcards for quick revision  
- Quiz mode for self-assessment  
- Search functionality  
- Review system for revisiting weak areas  
- Progress tracking (mastered, seen, unseen)  
- Streak tracking for consistency  

---

## Content Structure

- Multiple topics covering core data science concepts  
- Each topic contains a set of practice questions  
- Questions can be marked as:
  - Unseen  
  - Seen  
  - Mastered  

---

## Usage

- Select a topic and start practicing  
- Reveal answers and mark progress  
- Use flashcards for quick learning  
- Take quizzes to test knowledge  
- Review previously attempted questions  

---

## Goal

Provide a simple and effective way to practice and retain data science concepts through repetition and tracking.
